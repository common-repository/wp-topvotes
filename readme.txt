This Plugin let the visitor vote your site via a simple widget like the facebook "I like this" button.

== Description ==

This service allows your visitors to vote for your website. It is easy to integrate the service into your web pages. There is no own webspace needed.
Where's the difference?
The difference to other voting services is that every user can vote. The users do not need an account to submit their votes.

This section describes how to install the plugin and get it working.

   2. Activate the plugin through the ‘Plugins’ menu in WordPress
